# Warhammer 40k 11th Edition — Detachment & Mission Dashboard

Pick your faction, then set both players' force dispositions to see which
Chapter Approved 2026 primary mission each player plays. Clicking a detachment
row sets your disposition automatically.

A single static `index.html` — no build step, no dependencies. Open it locally
or serve it from any static host.

Data sources: Warhammer Community faction pack articles (June 2026) and the
Chapter Approved 2026 mission deck, cross-checked against goonhammer.com force
disposition reviews and gdmissions.app. Mission card images are served by
[gdmissions.app](https://gdmissions.app).
